# Идеи для будущих версий

## Голосовой ввод
- Кнопка 🎙️ рядом с полем ввода
- Варианты реализации:
  - **Локально (offline):** `speech_recognition` + `vosk` — бесплатно
  - **Облачно:** Whisper API (OpenAI) — лучшее качество, платно
- UX: зажал → записывает → отпустил → распознаёт → вставляет текст
- Сложность: 2-4 часа

## Автообновление
- Кнопка "Обновить" в сообщении чата о новой версии
- Варианты:
  1. Скачать инсталлятор → запустить → закрыть приложение (проще)
  2. Скачать ZIP → распаковать → перезапустить (сложнее, нужен updater)

## Telegram бот для обратной связи
- Бот для приёма замечаний, предложений, баг-репортов от пользователей
- В приложении: ссылка на бота (в настройках или меню трея)
- Реализация: python-telegram-bot или aiogram
- Сообщения пересылаются в личку/группу разработчика
